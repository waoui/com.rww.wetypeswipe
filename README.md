# 微信输入法下滑快捷键

为安卓版微信输入法增加自定义按键下滑操作的 LSPosed 模块。

支持 26 键与九宫格，可将全选、剪切、复制、粘贴、复制全部、剪切全部、段落定位、原生剪贴板和快捷发送绑定到指定按键。正常点击保持原有输入，下滑时执行快捷功能。

## 主要功能

- 全选、剪切、复制、粘贴
- 复制全部、剪切全部
- 移动到段首、段尾
- 选择当前位置到段首、段尾
- 打开微信输入法原生剪贴板
- 打开微信输入法原生快捷发送／常用语
- 禁用指定按键的下滑动作
- 支持 26 键和九宫格 2–9
- 支持按键底部功能标签和下滑触发提示
- 支持自定义或隐藏单个按键标签
- 支持竖屏、横屏、分离键盘及横屏悬浮键盘
- 26 键与九宫格可独立配置触发距离
- 原生剪贴板与快捷发送无需显示在输入法工具栏
- 数字、符号和手写键盘自动停用模块标签及下滑动作
- 密码输入框自动停用模块功能
- 不联网，不收集、上传或保存输入内容

## 安装与使用

1. 在 LSPosed 模块仓库中安装最新版。
2. 在 LSPosed 中启用本模块。
3. 将模块作用域勾选为“微信输入法”。
4. 强制停止并重新启动微信输入法，或重启设备。
5. 打开模块设置，配置按键并点击“保存并应用配置”。

## 默认映射

| 按键 | 下滑动作 |
| --- | --- |
| Z | 全选 |
| X | 剪切 |
| C | 复制 |
| V | 粘贴 |

## 兼容性

- Android 8.0 及以上
- LSPosed / Modern Xposed API
- 安卓版微信输入法
- 当前实测：微信输入法 3.5.0、3.5.2；Android 16

微信输入法升级后，内部实现变化可能影响原生面板功能。遇到问题时，请提供微信输入法版本、Android 版本、LSPosed 日志和复现步骤。

## 版本同步

正式版首先发布到源代码仓库，LSPosed 模块仓库每 15 分钟自动检查并同步最新正式 Release。测试版和预发布版不会进入 LSPosed 模块仓库。
